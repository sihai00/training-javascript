- 队列
  - 普通队列：先进先出，后进后出
  - 优先队列：出队顺序和入队顺序无关，和优先级有关。（排序后执行也可以达到效果，但如果是动态队列，那么非常耗时不现实）
- 堆：任何一个节点都不大于它的父亲节点
- 二分搜索树：当前节点比左孩子大，比右孩子小
- 并查集：可用数组表示，键：当前节点；值：指向父节点
- 邻接表：表示稀疏的图（边少）
- 邻接矩阵：表示稠密的图（边多）或者完全图（所有点都与所有点产生边）
